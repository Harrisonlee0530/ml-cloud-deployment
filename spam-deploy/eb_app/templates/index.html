Use it from the terminal

This app also exposes a JSON API endpoint, so you can call it with curl.

curl -X POST "{{ request.host_url.rstrip('/') }}/api/predict" \ -H "Content-Type: application/json" \ -d '{"message":"Congratulations! You have won a free ticket. Reply now!"}'

Expected JSON response:

{ "message": "Congratulations! You have won a free ticket. Reply now!", "prediction": "spam", "raw_result": { "predicted_labels": ["spam"] } }

The Random message button uses free public APIs to fetch a natural-language message and then classifies it.
